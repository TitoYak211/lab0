Our Program Topics for Introduction to Computer Science

During this accelerated Program, we will use proven and successful blended learning model, as well as the flipped classroom methodology. We plan to teach:

  1. HTML, CSS, and Git

    • Command line
    • Text editors
    • Version control and Git
    • HTML page layout elements
    • CSS basics
    • Styling with classes
    • Debugging CSS and HTML
    • GitHub pages
    • Responsive design
    • Media queries

  2. JavaScript Fundamentals

    • Introduction to JavaScript
    • JavaScript console
    • Fundamental JavaScript
    • Writing functions
    • Business vs User interface logic
    • Control flows
    • Logical operators
    • Arrays
    • Looping

  3. JavaScript for Web and Front-End Development

    • Bootstrap
    • Navigating bootstrap documentation
    • Introduction to JQuery
    • jQuery effects
    • DOM manipulation and traversal
    • Variable Scope
    • Forms
    • Debugging in JavaScript

  4. Object-Oriented Programming and Git Collaboration

    • Intro to object-oriented programming
    • Contractors and prototypes
    • Objects within objects
